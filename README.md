# ExerciseSchedule
First Year Database Programming Project

# Overview
This is first year a group project where we had to create an Exercise Schedule System applying all the knowledge gained from database programming (Tables, Procedures, Triggers, Cursors).

# Contribution
My contribution is creating the database using MSSQL Server.
The following tables were created: Users, Preferences, DietPlan, Settings, Notification, WorkoutPlan, User_Settings, User_Notification, User_Preference.
